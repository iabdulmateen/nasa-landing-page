# NASA Landing Page Clone 🚀

A high-performance, responsive clone of the official NASA landing page. This project focuses on modern UI/UX principles, clean code structure, and a mobile-first design approach.

## 🌟 Key Features
- **Fully Responsive:** Optimized for seamless viewing across mobile, tablet, and desktop devices.
- **Cinematic Hero Section:** Features high-resolution space imagery with smooth gradient overlays for better text readability.
- **Interactive Navigation:** Includes a functional mobile hamburger menu and a search interface.
- **NASA Brand Identity:** Strictly follows the official NASA color palette (Insignia Red, Black, and White).
- **Structured Data Grid:** A clean 3-column info grid for mission details and quick links, which stacks beautifully on mobile.

## 🛠️ Built With
* **HTML5:** For semantic and accessible web structure.
* **CSS3:** Advanced styling using Flexbox, CSS Grid, and Media Queries.
* **Bootstrap:** Used for rapid responsive component development.
* **Google Fonts:** Utilizing professional typography (Outfit/Inter) to match the space-tech aesthetic.
